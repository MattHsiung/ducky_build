If you plan on streaming please a take a moment to read through the following information to help you connect to our services and get streaming some code!

Download the module!

npm install -g ducky-stream

Ducky-stream is the node module for ducky. This module allows you to stream live file updates to your channel.

Start streaming!

Run ducky login and enter your credentials. The ducky module will provide you with a secret key that you can then use to stream.

Add this key as the stream key in your video streaming application, such as OBS.

Next run ducky stream from the directory you want to share to begin streaming your file system.

Usage

Ducky will watch your file system from the directory you run ducky stream in. This sends up the contents of your files to then be displayed on your channel.

Ducky will not share any files that you have included in your .gitignore.
