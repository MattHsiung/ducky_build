Download the module!

npm install -g ducky-stream

Ducky-stream is the node module for ducky. This module allows you to stream live file updates to your channel.

Installation

npm install -g ducky-stream. Run ducky login and enter your credentials. The ducky module will provide you with secret key that you can use to stream. Add this key as the stream key in streaming video applications like OBS and start streaming. Run ducky stream from the directory you want to share to begin streaming your file system. Usage Ducky will watch your file system from the directory you run ducky stream in. This sends up files that are shown to the audience watching your livestream on www.ducky.com/channels.
Ducky will not share any files that you have included in your .gitignore.
